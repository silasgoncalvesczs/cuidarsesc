Backup e Restauração

Conta

Gerar backup

Inclui usuários, participantes (com biometria), atividades e presenças. Ao baixar, você escolhe a pasta e o nome do arquivo no seu computador.

Restaurar backup

Substitui os dados atuais pelos do arquivo. Esta ação não pode ser desfeita. Faça um backup recente antes de restaurar.

Automação

Os backups manuais já permitem escolher onde salvar no PC. Para agendar (diário/semanal), use a pasta do servidor e um agendador do Linux: 

docker compose exec app node src/scripts/backupLocal.js

Exemplo de cron (todo dia às 2h), na máquina host, com o projeto como pasta atual.
